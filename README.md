## Getting Started

1. **Clone the repository**
    ```bash
    git clone https://github.com/YOUR-USERNAME/Uni-Projects.git
    cd Uni-Projects
    ```

2. **Run any project**
    ```bash
    python3 <project_name>.py
    ```
    _Some projects may require additional packages. Check the top of each file for notes._
